# Список кредиторов — инструкция

## Что это

Программа автоматически собирает **список кредиторов для заявления о банкротстве** из двух документов:
1. Кредитная история из НБКИ (PDF)
2. Выписка исполнительных производств с Госуслуг (DOCX)

На выходе — готовый Word-документ с заполненными Разделом I и Разделом II.

---

## Установка (один раз)

1. Скачайте файл **`Список_кредиторов.exe`**
2. Положите его в любую папку (например, на Рабочий стол)
3. Готово. Никаких других программ устанавливать не нужно

> Программа полностью локальная — никакие данные никуда не отправляются. Всё обрабатывается на вашем компьютере.

---

## Как пользоваться

### Шаг 1. Запустите программу
Двойной клик на `Список_кредиторов.exe`

Откроется чёрное окно с логами и **автоматически запустится браузер** на странице приложения.

> ⚠️ Чёрное окно НЕ закрывайте, пока работаете с программой — это и есть сама программа. Закроете окно — программа выключится.

### Шаг 2. Загрузите документы
В браузере появится форма с двумя полями:

- **Кредитная история НБКИ** — выберите свой PDF-файл
- **Исполнительные производства** — выберите DOCX из Госуслуг

### Шаг 3. Нажмите «Создать документ»
Через 1-2 секунды браузер автоматически скачает готовый файл `creditors_list.docx` в папку «Загрузки».

### Шаг 4. Откройте результат в Word
Проверьте документ. **Обязательно вручную заполните:**
- Адреса кредиторов (берутся из ЕГРЮЛ по ИНН)
- Кредиторы, которых нет в НБКИ или ИП (например, иски по ДТП)
- Исторические налоговые ИП (если их нет в текущей выгрузке Госуслуг)

### Шаг 5. Закройте программу
Просто закройте чёрное окно — программа выключится.

---

## Что программа делает автоматически

✓ Извлекает кредиторов из НБКИ
✓ Учитывает переуступки долга (МСТ МОСТ вместо ПКО СОЛ и т.п.)
✓ Связывает кредиторов с исполнительными производствами
✓ Заполняет Раздел I (кредиты) и Раздел II (налоги, штрафы, исп.сборы)
✓ Сохраняет форматирование шаблона

## Что нужно проверить вручную

⚠ Адреса кредиторов
⚠ Кредиторы из других источников (иски, не попавшие в НБКИ)
⚠ Исторические ИП до 2022 года (могут отсутствовать в текущей выгрузке Госуслуг)
⚠ Суммы долга могут отличаться от вашего источника на 100-1000 рублей — НБКИ обновляется не сразу

## Если что-то не работает

**Программа не запускается / Windows ругается на безопасность:**
Правый клик на файле → Свойства → внизу галочка «Разблокировать» → ОК

**Браузер не открывается автоматически:**
Откройте вручную: http://localhost:5001

**Документ не скачался:**
Проверьте, что в чёрном окне нет ошибок. Если есть — пришлите скриншот.

**Программа закрылась с ошибкой:**
Сделайте скриншот сообщения и пришлите.
